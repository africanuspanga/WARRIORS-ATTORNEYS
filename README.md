# Warriors Attorneys Website

A premium, modern website for Warriors Attorneys, a Tanzanian law firm.

## Tech Stack

- Next.js 16.2
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

The static export is generated in the `dist/` directory.

## Project Structure

- `app/` — Next.js App Router pages
- `components/` — React components and shadcn/ui components
- `lib/data.ts` — Services, testimonials, and contact information
- `public/` — Images, logo, favicon, and assets
